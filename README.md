## How to Run
This is an Android project built using Jetpack Compose. To run it:

1. Clone this repo using `git clone` or the GitHub Desktop app.
2. Open **Android Studio** (If you don't have it, download it from [developer.android.com/studio](https://developer.android.com/studio)).
3. Select **Open** and navigate to the directory where you cloned this repository.
4. Wait for Android Studio to finish syncing the Gradle build (you'll see a progress bar at the bottom).
5. Click the green **Run** button (▶) at the top to launch the app on an emulator or a connected physical device.
